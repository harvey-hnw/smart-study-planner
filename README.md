# Smart Study Planner

A simple web app to plan weekly classes, detect schedule clashes, and visualize courses in a timetable grid.

## Features
- Add courses with day, start, and end time
- Clash detection for overlapping classes
- Delete individual courses / clear all
- Weekly timetable view
- Data saved in browser localStorage

## Run
Open `index.html` directly, or run a local server:
